Class: CountedList

CountedList(objs, source)

Generic list-ish class for NamedTootCount objects with 'name' and 'numToots' properties. Supports both dictionary and sorted list operations, and provides utility methods for filtering, mapping, counting, and muting/removing items by keywords or server-side filters.

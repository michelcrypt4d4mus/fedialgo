Members

ALL_CACHE_KEYS

Array of all cache keys (CacheKey and TagTootsCacheKey values).

AlgorithmStorageKey

Enum of storage keys for user data and app state (not API cache).

BooleanFilterName

Enum of boolean filter names for filtering toots by property.

CacheKey

Enum of keys used to cache Mastodon API data in the browser's IndexedDB via localForage. Keys that contain Toots should end with "_TOOTS", likewise for Account objects with "_ACCOUNTS". Used for Storage and cache management.

MediaCategory

Enum of Mastodon API media category strings.

NonScoreWeightName

Enum of non-score weight names (used for sliders and scoring adjustments). Order influences the order of the score weighting sliders in the demo app.

ScoreName

Enum of all scoring categories for which there is a scorer. Also Used for UI display and filtering.

TagTootsCacheKey

Enum of cache keys for hashtag-related Toot lists.

TrendingType

Enum of trending data types that can be fetched from the API.

TypeFilterName

Enum of type filter names for filtering toots by type (e.g., audio, bot, images, etc.). The values have spaces for better presentation in the demo app.

Class: ObjWithCountList

ObjWithCountList~ObjWithCountList()

A generic list class for objects with a name and a 'numToots' property. Supports both dictionary and sorted list operations, and provides utility methods for filtering, mapping, counting, and muting/removing items by keywords or server-side filters.
